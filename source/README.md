# Enterprise B2B Commerce Runtime

Frontend architecture scaffold for a stateful mobile B2B commerce runtime.

## Stack
- React
- TypeScript
- Vite
- Zustand
- TanStack Query
- React Router
- TailwindCSS
- Framer Motion

## Notes
- Frontend is render-only.
- Pricing and inventory remain authoritative outside the UI.
- This scaffold uses a mock gateway layer to remain runnable without a backend.
